# QR Code WebApp

A modern QR code generator webapp with multi-language support, built with HTML, CSS, and JavaScript.

## Features
- Generate QR codes for URLs, text, phone numbers, and emails
- QR code design with rounded corners and dots
- HD SVG output for crisp quality
- Download branded QR code images (1080x1080px)
- Multi-language support (English/Italian)
- Clean and responsive UI
- Mobile and tablet optimized
